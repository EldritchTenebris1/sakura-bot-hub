

## Sakura Bot — Site de Documentação 🌸

### Estilo Visual
- Tema escuro com detalhes em rosa/sakura
- Gradientes sutis com tons de rosa e magenta
- Animações suaves de pétalas de cerejeira como decoração
- Fontes modernas e elegantes

### Páginas

**1. Landing Page (Página Inicial)**
- Hero com nome "Sakura Bot", descrição curta e logo/ícone
- Botões: "Ver Documentação", "Comprar no BuiltByBit", "Discord de Suporte"
- Seção de destaques das funcionalidades (Moderação, Economia, Tickets, Boas-vindas)
- Estatísticas ou badges visuais
- Footer com links

**2. Documentação (com sidebar de navegação)**
- **Moderação**: Ban, kick, mute, warn, logs
- **Economia/Levels**: Sistema de XP, níveis, moedas, loja
- **Tickets/Suporte**: Sistema de tickets, formulários
- **Boas-vindas/Saídas**: Mensagens automáticas, autorole
- Cada comando com: nome, descrição, uso, permissões e exemplo

**3. Comandos**
- Lista pesquisável de todos os comandos
- Filtro por categoria
- Formato visual tipo "cards" com prefixo, uso e permissão

**4. Changelog (Atualizações)**
- Lista de versões com data e mudanças
- Tags: novo, correção, melhoria

**5. FAQ**
- Perguntas e respostas em formato accordion
- Perguntas comuns sobre instalação, configuração, erros

### Navegação
- Header fixo com logo, links para cada página, botão do Discord e BuiltByBit
- Sidebar na documentação para navegar entre categorias
- Responsivo para mobile com menu hamburger
